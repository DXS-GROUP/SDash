<h3>Installing</h3>

```
git clone https://github.com/Nighty3098/ServerMonitor
cd ServerMonitor
python3 -m venv monitoring
source monitoring/bin/activate
pip3 install -r requirements.txt
```

<h3>Run server</h3>

```
python3 app.py
```
