  <h3>Installing</h3>
  
  ```
  git clone https://github.com/Nighty3098/SDash
  cd SDash
  python3 -m venv SDash
  source SDash/bin/activate
  pip3 install -r requirements.txt
  ```
  
  <h3>Run server</h3>
  
  ```
  python3 server.py
  ```
