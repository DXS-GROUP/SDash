  <h3>Installing</h3>
  <div align="left">
    
  ```
  git clone https://github.com/SDashS/SDash
  cd SDash
  python3 -m venv SDash
  source SDash/bin/activate
  pip3 install -r requirements.txt
  ```
  </div>
  
  <h3>Run server</h3>
</div>

> [!IMPORTANT]
> This service has been tested on Arch, Debian, Ubuntu, Fedora based systems

```
python3 server.py
```
